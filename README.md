# 📊 RetailPulse

## AI-Powered Customer Analytics & Demand Forecasting Platform

RetailPulse is an end-to-end retail analytics platform developed using **Python, Machine Learning, and Streamlit**. The project transforms raw retail transaction data into actionable business insights by combining data analytics, predictive modeling, inventory optimization, and interactive dashboard visualization.

The platform helps businesses understand customer behavior, forecast future sales, optimize inventory levels, and identify customers who are at risk of churning. It provides an easy-to-use dashboard for business users to monitor key performance indicators (KPIs) and make data-driven decisions.

## 🚀 Project Objectives

- Clean and preprocess retail transaction data.
- Perform Exploratory Data Analysis (EDA).
- Segment customers using RFM Analysis.
- Forecast future sales using Prophet and LSTM.
- Predict customer churn using XGBoost.
- Optimize inventory using EOQ, Safety Stock, and Reorder Point.
- Develop an interactive Streamlit dashboard.
- Generate downloadable business reports.

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/Anuraagh20L/RetailPulse.git
```

Navigate to the project directory:

```bash
cd RetailPulse
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
